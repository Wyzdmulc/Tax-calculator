# Tax Calculator (Malawi)

A simple web-based tax calculator built with **HTML, CSS, and JavaScript**.  
It helps users in Malawi calculate and understand the correct amounts payable to customers and authorities based on different tax scenarios.

---

## âœ¨ Features
- Calculates **VAT (17.5%)**, **PPDA (1%)**, **NCIC (1%)**, **Tourism Levy (1%)**, and **Withholding Tax (WHT)**.
- Handles multiple scenarios:
  - VAT only
  - PPDA only
  - VAT + PPDA
  - VAT + PPDA + NCIC
  - VAT + Tourism Levy
  - VAT + PPDA + Tourism Levy
  - None of the above
- Professional design inspired by **Malawi Revenue Authority (MRA)** color palette.
- Responsive layout for desktop and mobile.
